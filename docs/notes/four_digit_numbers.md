# Chinese Four-Digit Numbers

## Summary

* Four-digit numbers are built with **千** (*qiān*, thousand).
* The basic pattern is **X千Y百Z十W**.
* When one internal place is missing, Chinese normally inserts **零**.
* Multiple missing places are usually represented by **one 零**, not repeated **零零**.
* Inside number names, use **二**, not **两**, in the regular structural form.
* The main learner difficulty is knowing **when 零 is required** and **when it is not**.

## Core pattern

The core scheme is:

* **1000**: **一千**
* **1001**: **一千零一**
* **1010**: **一千零一十**
* **1100**: **一千一百**
* **1234**: **一千二百三十四**
* **9009**: **九千零九**

Canonical patterns:

* **一千**
* **一千零五**
* **一千零一十**
* **一千一百零二**
* **三千零八十**
* **九千九百九十九**

## Explanation

### 1. The basic structure

Four-digit numbers use **千** for “thousand.”

A full four-digit number is:

**thousands digit + 千 + hundreds digit + 百 + tens digit + 十 + ones digit**

Example:

* **三千二百四十五** = 3245

### 2. No missing place: just read each value in order

When all places are present, the pattern is straightforward.

* **1234** = **一千二百三十四**
* **5678** = **五千六百七十八**

There is no **零** because no internal place is missing.

### 3. Missing hundreds, tens, or ones

If a place in the middle is missing, Chinese usually inserts **零** once to mark the gap.

* **1001** = **一千零一**
* **1010** = **一千零一十**
* **1101** = **一千一百零一**
* **1050** = **一千零五十**

The rule is practical, not mechanical letter-by-letter reading: **零 marks a gap inside the number**.

### 4. One 零 can cover more than one missing place

If both the hundreds and tens places are missing, Chinese still usually uses only one **零**.

* **1001** = **一千零一**

* not **一千零零一**

* **3008** = **三千零八**

* not **三千零零八**

This is one of the most important points.

### 5. Do not add 零 at the end

If the number ends in zero, you normally stop at the last nonzero place.

* **1200** = **一千二百**
* **3400** = **三千四百**
* **5060** = **五千零六十**
* **7000** = **七千**

So:

* internal missing place → often needs **零**
* final empty place(s) → no final **零**

### 6. Numbers from 1000 to 1999

In careful standard Mandarin, **1000** is **一千**, not just **千**.

Likewise:

* **1100** = **一千一百**
* **1010** = **一千零一十**

For learners, always include the initial **一**.

### 7. 二, not 两, in number formation

In systematic numeral building, use **二** inside number names:

* **二千** = 2000
* **二千零二十** = 2020
* **二千二百二十二** = 2222

In ordinary speech, **两千** is also common when referring to quantities. But for a structure-based number page, **二千** is the safest standard form.

!!! note
In everyday speech, forms like **两千** and **两百** are very common for quantities. In a number-formation note, teach **二千** and **二百** as the regular structural forms.

## Examples

!!! example "Example 1"
一千
yìqiān
one thousand

```
This is the basic form for 1000.
```

!!! example "Example 2"
一千零三
yìqiān líng sān
one thousand and three

```
**零** marks the missing hundreds and tens section.
```

!!! example "Example 3"
一千零二十
yìqiān líng èrshí
one thousand and twenty

```
The hundreds place is missing, so **零** is required.
```

!!! example "Example 4"
一千一百
yìqiān yìbǎi
one thousand one hundred

```
There is no missing internal place before **百**, so no **零** appears.
```

!!! example "Example 5"
一千一百零五
yìqiān yìbǎi líng wǔ
one thousand one hundred and five

```
The tens place is missing, so insert **零**.
```

!!! example "Example 6"
一千二百三十四
yìqiān èrbǎi sānshísì
one thousand two hundred and thirty-four

```
Full pattern: **thousands + hundreds + tens + ones**.
```

!!! example "Example 7"
三千零八十
sānqiān líng bāshí
three thousand and eighty

```
The hundreds place is missing.
```

!!! example "Example 8"
九千九百九十九
jiǔqiān jiǔbǎi jiǔshíjiǔ
nine thousand nine hundred and ninety-nine

```
This is the highest four-digit number.
```

## Common errors / pitfalls

* Omitting **零** when there is an internal gap.

  * **1005** = **一千零五**, not **一千五**
  * **3050** = **三千零五十**, not **三千五十**

* Repeating **零** for each missing place.

  * **1001** = **一千零一**, not **一千零零一**

* Adding **零** at the end.

  * **1200** = **一千二百**, not **一千二百零**

* Using **两千** as the structural form in a number-building table.
  For regular numeral formation, **二千** is the safer standard form.

* Dropping **一** in numbers like 1100.

  * **1100** = **一千一百**, not **一千百**

!!! warning
The key rule is this: use **零** to mark an internal missing place, but do not use repeated **零零** and do not add **零** at the end.

## Comparison

### 1. 一千零五 vs 一千五百

* **一千零五** = 1005
* **一千五百** = 1500

The first has missing hundreds and tens, so it needs **零**. The second has a real hundreds digit.

### 2. 一千零一十 vs 一千一百

* **一千零一十** = 1010
* **一千一百** = 1100

These differ in which place is occupied after **千**.

### 3. 二千 vs 两千

* **二千** = standard structural numeral form
* **两千** = common in everyday quantity expressions

For a structure-based page, teach **二千** first.

## Notes

!!! tip
Build four-digit numbers one place at a time: thousands, hundreds, tens, ones. Insert **零** only where a non-final place is missing before a later nonzero digit appears.

!!! note
In real usage, speakers may prefer **两千**, **两百**, etc. in quantity expressions. For systematic number formation, **二千**, **二百** give the clearest pattern.

!!! note
For numbers above 9999, Chinese continues with larger units such as **万**. Four-digit numbers are the last range that can be handled cleanly without introducing that next unit.
