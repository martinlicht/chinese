# Teach Yourself Chinese

These are notes for teaching yourself Chinese, written in English. 

This is material is mostly AI-generated and covers basic topics. I am setting it up primarily for myself, but I hope other people may find it useful as well. 

The HTML is automatically generated from Markdown files. 

## Goals
 - self-contained presentation of basic material
 - both Hanzi and Pinyin
 - vocabulary, basic phrases, some grammar
 - for engineers, researchers, scientists

## Caveats 
 - This notes will not go into the depth of every topic
 - Speaking, hearing, and stroke orders are not covered
 - Being AI-generated, no correctness is guaranteed
